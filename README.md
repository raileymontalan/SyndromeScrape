# SyndromeScrape

### Before running the server
```
$ mongo
$ use articles
```

### To run the server
For Mac:
```
$ python3 server.py
```
For Windows:
```
$ python server.py
```

### To run webscraping
For Mac:
```
$ python3 scrape.py
```
For Windows:
```
$ python scrape_win.py
```

---

Note: Static files (TXT, JSON) are located in */static* directory.
